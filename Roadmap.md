# Roadmap

## MVP

- SwiftData local subscription storage.
- Calendar grid with payment indicators.
- Subscription list with filters, sorting, search, edit, archive, and delete.
- Subscription editor.
- Category analytics with Charts.
- Notification scheduling and test notifications.

## Next

- Import and export CSV.
- Backup history and one-click restore points.
- Better custom recurrence rules.
- Budget limits and over-budget warnings.
- Menu bar upcoming-renewals view.
- iCloud sync.
- Optional menu bar upcoming-renewals widget.
- Shared code target for future iOS app.

## Later

- Signed and notarized installer workflow.
- Optional cloud backup snapshots.
- Multi-currency conversion with user-provided rates.
- Widgets and Shortcuts.
