# Prompt для Codex — macOS приложение Subscription Calendar

Ты — senior macOS developer и product designer. Создай нативное macOS-приложение на SwiftUI, вдохновленное идеей Subscription Day Calendar, но с оригинальным названием, дизайном и кодом. Не копируй чужие ассеты, тексты, иконки или точную верстку. Сделай самостоятельный продукт для учета подписок и регулярных платежей.

## Название проекта
SubPulse

## Цель
Создать красивое, минималистичное macOS-приложение для учета подписок:
- календарь платежей;
- сумма расходов за месяц;
- список активных подписок;
- категории;
- способы оплаты;
- аналитика;
- напоминания;
- локальное хранение данных;
- возможность собрать `.app` и потом упаковать в `.dmg`.

## Технологии
Используй:
- Swift
- SwiftUI
- SwiftData
- MVVM
- UserNotifications
- Charts
- SF Symbols
- macOS 14+

## Основные экраны

### 1. DashboardView
Главный экран:
- сверху заголовок `All Subs`;
- справа кнопки поиска, статистики и настроек;
- в центре месяц и год;
- крупная сумма расходов за текущий месяц;
- подпись типа `Regular Month`;
- календарная сетка месяца;
- на днях оплаты показывать маленькие иконки подписок;
- внизу кнопка `Add Subscription`;
- рядом бейдж с оставшимся лимитом для free-версии.

### 2. CalendarView
- сетка дней месяца;
- отображение подписок на конкретных датах;
- выбор дня;
- список платежей выбранного дня;
- переключение месяца назад / вперед.

### 3. SubscriptionListView
- список всех подписок;
- фильтры: active / archived / all;
- сортировка: дата, сумма, название;
- поиск;
- быстрые действия: edit, archive, delete.

### 4. SubscriptionEditorView
Форма добавления / редактирования:
- название;
- сумма;
- валюта;
- дата следующего платежа;
- период: weekly / monthly / yearly / custom;
- категория;
- способ оплаты;
- иконка;
- заметка;
- переключатель активна / неактивна;
- кнопка save.

### 5. AnalyticsView
- кольцевая диаграмма по категориям;
- годовой прогноз;
- средняя месячная стоимость;
- количество активных подписок;
- фильтр по году;
- фильтр по категории.

### 6. SettingsView
- тип аккаунта Free / Pro;
- iCloud & Data placeholder;
- основная валюта;
- округление;
- compact numbers;
- категории;
- списки;
- способы оплаты;
- первое напоминание;
- второе напоминание;
- тестовое уведомление;
- appearance: system / light / dark;
- haptic feedback placeholder;
- about section.

## Логика
- Корректно рассчитывай платежи текущего месяца.
- Учитывай weekly / monthly / yearly подписки.
- Показывай только активные подписки в месячной сумме.
- Годовой прогноз считай по всем активным подпискам.
- Среднюю месячную стоимость считай как yearlyForecast / 12.
- Добавь seed demo data при первом запуске: iCloud+, Google One, ChatGPT.

## UI стиль
- Нативный macOS SwiftUI.
- Чистый минимализм.
- RoundedRectangle с мягкими скруглениями.
- Системные цвета.
- Поддержка light/dark mode.
- Аккуратные SF Symbols.
- Большие цифры на главном экране.
- Плавные анимации.
- Хорошая адаптация окна от 900x650 и выше.
